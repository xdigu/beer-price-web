# beer-price-web

A simple way to know witch beer is the cheaper one.

## How to run this project

Clone this repo and enter it:

```sh
$ git clone https://github.com/xdigu/beer-price-web.git
$ cd beer-price-web
```

Install dependencies:

```sh
$ yarn install
```

Now you can run:

```sh
$ yarn start
```

or with docker:

```sh
$ docker-compose up
```
